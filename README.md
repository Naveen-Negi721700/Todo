# iTask - Todo Application

A simple Todo application built using **React** and **Tailwind CSS** that helps users manage their daily tasks efficiently.
The application allows users to add, edit, delete, and mark tasks as completed. It also stores todos in **Local Storage**, so tasks remain saved even after refreshing the page.

## Technologies Used

React.js
Tailwind CSS
JavaScript
Vite

## Features

Add new todos
Edit existing todos
Delete tasks
Mark tasks as completed
Show or hide finished tasks
Local storage support
